# `@masknet/type/no-const-enum`

Disallow use constants enumerate

<!-- end title -->

## Rule Details

### :x: Incorrect

```ts
const enum Foo {
  Bar,
}
```

### :white_check_mark: Correct

```ts
enum Foo {
  Bar,
}
```
