# `@dimensiondev/jsx/no-set-html`

Disallow use `dangerouslySetInnerHTML` jsx attribute

<!-- end title -->

## Rule Details

## Options

### :x: Incorrect

```jsx
<a dangerouslySetInnerHTML={{ __html: '....' }} />
```
